### **Thermal Printer Commands Used:**

| Command | Purpose |
|---------|---------|
| `\x1ba\x01` | Center align text |
| `\x1ba\x00` | Left align text |
| `\x1b!\x18` | Bold text (double height/width) |
| `\x1b!\x00` | Reset text formatting |
| `\x1dV\x42\x00` | Paper cut command |

### **Line Width:**
- Standard thermal printer: **42 characters**
- Services formatted using `formatLine()` function
- Automatic text wrapping for long service names
